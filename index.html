地面に置かれたARの仮想ボールを、実際に足で蹴ろう。蹴るフォーム・ボールへのインパクトポイント・スイングの速さで、シュートの方向と質が変わります。

📱 はじめかた
  1. スマホを床や台に立てかけて固定する(横画面推奨)
  2. 2〜3歩下がり、足元まで全身がカメラに映る位置に立つ
  3. 足が検出されると、あなたの足元にARボールが置かれる
  4. 助走をつけて、画面(=ゴール)に向かって本気で蹴る!
🥅 PKチャレンジのルール
  • カメラはゴール裏視点 — 画面下のゴールへ蹴り込む
  • 1セット5本。キーパーは蹴った瞬間にどこかへ飛ぶ
  • コーナー(左右の隅)ほど高得点。上隅は最高難度で最高得点
  • 正面に蹴るとセーブされる — キーパーの逆を突こう
  • 強烈なシュートは、キーパーが触れても弾き込めることがある
🌀 フリーキックのルール
  • ボールとゴールの間に壁(ジャンプする)が立つ
  • 壁を破る方法は2つ — 浮かせて越える or カーブで巻く
  • ボールの下側を蹴ると浮く。インフロントで蹴ると曲がる
  • 距離が遠いぶんキーパーの反応時間も長い。コースの厳しさが命
  • 壁越え・カーブ弾にはボーナス得点
🦵 蹴り方で変わるシュート
  • スイングの速さ → シュートの威力。速いほど強く、キーパーが反応しづらい
  • 振り抜く方向 → シュートの方向。蹴りたいコースへフォロースルー
  • インパクトポイント → ボールの下側を蹴ると浮き、真ん中〜上だとグラウンダー。左右にずらすと逆側へ飛ぶ
  • インステップ(足の甲) → 最大威力のストレート
  • インサイド(足の内側) → 威力は控えめでも正確。PK向き
  • インフロント(親指の付け根) → カーブがかかる。FK向き
  • アウトフロント → 逆回転のカーブ(アウトスイング)
  • 強振しすぎ+ボールの下すぎ = ふかして枠上に注意
🏆 得点の早見表
  • ゴール 100点 + コース加点(サイド+30 / 上段+40)
  • キーパーの逆を突く +30 / 強シュート +20
  • 高速スイング PERFECT +20
  • FK: 壁越えゴール +50 / カーブ弾 +60
  • ポスト直撃からのゴールイン 「ポストイン!」+20
