📱 セッティングと安全
  1. スマホを倒れないよう固定し、全身と両足が映るまで2〜3歩下がる。
  2. 横画面推奨。周囲2m以上を空け、家具・人・ペットがいないことを確認する。
  3. 実物のボールは置かず、画面上の仮想ボールに向けて空振りする。
  4. インカメラは画面を見ながらプレー可能。アウトカメラは補助者がいる時に推奨。
🦵 ボールが変化する仕組み
  • 振り抜き方向＋軸足の向き → 左右のコース。
  • ボール下部へのインパクト＋上向きスイング → 高さと壁越え。
  • インサイド／アウトサイド → カーブ方向と回転量。
  • スイング速度＋フォーム品質 → 初速、キーパーの届きやすさ、得点。
